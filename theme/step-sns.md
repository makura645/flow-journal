# 歩数課金SNS（Nostr版）

## コンセプト
いいねに歩数を消費する分散型SNS

---

## コア機能

### 1. 歩数課金システム
- 100歩 = 1いいね
- 500歩 = 1リポスト
- 歩数残高が可視化

### 2. Nostr統合
- 通常のNostrクライアントとして動作
- リアクション（kind: 7）に歩数メタデータを追加
- リレーに歩数付きイベントを配信

### 3. 歩数トラッキング
- Web API (navigator.permissions.query)
- 手動入力（ハッカソン用）
- フィットネスアプリ連携（将来）

---

## Nostrイベント構造

```json
{
  "kind": 7,
  "content": "🚶",
  "tags": [
    ["e", "投稿ID"],
    ["p", "投稿者公開鍵"],
    ["steps", "500"]
  ]
}
```

---

## デモの流れ

1. 歩数入力（例: 10000歩）
2. タイムライン表示
3. いいね → 100歩消費 → 残り9900歩
4. 「この投稿に500歩使った」バッジ表示

---

## ピッチ

**課題**: SNSで無限に消費されるリアクション、価値が見えない

**解決**: 歩数という「体を動かした証明」でリアクションに価値を

**価値**: 「この投稿に500歩も使った」= 本当に響いた証
