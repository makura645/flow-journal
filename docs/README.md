## 📊 プロジェクト概要

- **イベント**: バイブコーディング大会
- **テーマ**: テキストエディタ x タイピング速度
- **コンセプト**: 止まると霞む、思考の流れを途切れさせないジャーナリングツール
- **開発期間**: 1日
- **技術スタック**: React / Vue / Vanilla JS + CPM計測 + CSS

詳細は `/CLAUDE.md` と `/README.md` を参照
