# ユーザーストーリー

実際の使用シーンを想定したUX

---

## 1. 朝のモーニングページ（15分）

シーン: 昨日の振り返り、今日の予定を書く

手が止まる場面:
- 昨日の出来事を一覧にした後
- 今日の優先順位を考えるタイミング

ゲージの動き:
- 190-230CPMで安定、ゲージ70%維持
- 迷い始めると120CPMに落ち、じわじわ減少

文字消滅寸前:
- アイデアが浮かばずゲージ0% → 2秒カウント
- 「朝の気持ち」を箇条書きで高速入力
- 2秒で80字達成 → グレーアウト解除

終了時の気づき:
- Flow Scoreが高得点
- 昨日→今日のつながりを可視化
- 「迷った瞬間でもペンを止めずに次のテーマを掘れる」感覚

---

## 2. 悩み事の整理（自由時間）

シーン: 仕事のモヤモヤを言語化したい

手が止まる場面:
- モヤモヤの原因を深掘りしようとする
- 感情が重くなり、入力が途切れる

ゲージの動き:
- 序盤は200CPMで順調
- 原因を書こうとすると100CPM以下に急落
- 停止が続くとゲージほぼ0に

文字消滅寸前:
- グレー化が始まる
- 「なぜモヤモヤする？」と自問して感情を書き出す → 復活
- 1回失敗、直近2文が消滅、Flow Streakリセット
- 回復速度半減のペナルティ
- 「考えすぎて止まると記録が消える」緊張感で、感情を途切れさせず綴る

終了時の気づき:
- 感情語70% / 行動語30%
- 「行動の具体化を次セッションで」ガイド
- 止まらず書いた区間が感情処理に役立ったと実感

---

## 3. 感情の吐き出し（緊急時）

シーン: イライラして即座に書きたい

手が止まる場面:
- イライラの原因を一気に書く
- 途中で怒りが再燃し、考えすぎて停止

ゲージの動き:
- 開始直後は250CPM超でゲージMAX近く
- 怒りが頂点に達した瞬間に完全停止 → 即0%

文字消滅寸前:
- グレーアウト2秒で焦りを誘発
- 「今感じている身体感覚」「最悪の想定」をそのまま書き殴る
- 復活条件（2秒80字）をギリギリ達成
- 「止まらず吐き出す」ことで感情が整理される

終了時の気づき:
- Flow Scoreは中程度
- ピークCPM、停止回数、感情語の割合を可視化
- 怒りを内面化しながらも手を止めずに書いた区間を分析
- 緊急時でも「止まったら消える」仕組みが即時アウトプットを引き出した
- 科学的ベストプラクティス（連続書記）に沿った感情処理ができたと実感

---

## 共通の学び

止まらずに書くことの価値:
- 迷った瞬間でも次のテーマを掘れる
- 感情処理が深まる
- 考えすぎを防ぐ
- 即時アウトプットが促される

ゲージの役割:
- 思考の流れを可視化
- 止まりそうな時に警告
- 復活の達成感を与える

終了時の統計:
- 自分の書き方のパターンを知る
- 次回への改善ヒント
- 科学的根拠に基づく効果を実感
